**Explain YAML.**


"YAML (YAML Ain't Markup Language) is a human-readable data serialization language used to define configuration and automation in tools like Ansible, Kubernetes, and Docker Compose. It uses indentation instead of brackets or braces to represent data, making it easy to read and write. In Ansible, playbooks are written in YAML to describe hosts, tasks, variables, and modules."

Key Features
Human-readable and easy to write.
Uses spaces for indentation (not tabs).
Stores data as key-value pairs, lists, and dictionaries.
Widely used in Ansible, Kubernetes, Docker Compose, and CI/CD pipelines.
