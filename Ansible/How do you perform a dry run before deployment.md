**How do you perform a dry run before deployment?**


Before deploying changes, I perform a dry run using Ansible's check mode. Check mode simulates the execution of a playbook without making any actual changes, allowing me to verify what would happen. I also use the diff option to see the exact configuration changes that would be applied. This helps identify potential issues before the actual deployment."
