**tar instead of cp**


tar archives an entire directory structure into a single file.
It preserves:
Directory hierarchy
File permissions
Ownership (when applicable)
Timestamps
With -z, it also compresses the archive to save disk space.


**1. What does .tar.gz mean?**

.tar = archive of multiple files/directories.
.gz = compressed using gzip.

So .tar.gz is a compressed archive.
