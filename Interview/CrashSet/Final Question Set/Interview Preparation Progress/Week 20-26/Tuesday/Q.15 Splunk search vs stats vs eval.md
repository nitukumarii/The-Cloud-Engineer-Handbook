# Splunk Search vs Stats vs Eval

- **Search:** Used to find/filter specific events or logs based on conditions.  
  Example: Find all failed transactions.

- **Stats:** Used to perform calculations and summarize data.  
  Example: Count errors, calculate average response time, group results by service.

- **Eval:** Used to create or modify fields using expressions.  
  Example: Create a new field to classify errors or calculate values.
