# Terraform Workflow

Terraform workflow consists of:

1. **Terraform Init**
   - Initializes the Terraform working directory and downloads required providers.

2. **Terraform Plan**
   - Shows the changes Terraform will make before applying them.

3. **Terraform Apply**
   - Creates or updates the infrastructure resources.

4. **Terraform Destroy**
   - Removes the infrastructure resources when they are no longer required.
