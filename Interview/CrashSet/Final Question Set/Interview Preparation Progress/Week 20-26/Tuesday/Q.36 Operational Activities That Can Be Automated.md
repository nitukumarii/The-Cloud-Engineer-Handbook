# Operational Activities That Can Be Automated

Any repetitive manual task that consumes operational time can be automated.

Examples:

- **Patching and package updates** across multiple servers using Ansible.
- **Configuration management** such as updating configuration files and managing services.
- **Infrastructure provisioning** using Terraform to create cloud resources.
- **Backup automation** through scheduled jobs.
- **Log management** such as log rotation, cleanup, and archival.
- **Application deployments** using CI/CD pipelines.
- **Health checks and monitoring alerts** to detect issues automatically.

Automation helps reduce manual effort, minimize human errors, and reduce SRE toil.
