# 🔐 Security / DevSecOps Interview Questions

1. Describe your experience working with security teams during certificate renewals, vulnerability remediation, or production changes.

2. Can you outline your experience with security on AWS at a high level?

3. What are the AWS security best practices?

4. Have you ever worked on securing a web application hosted on AWS?

5. Can you walk us through a web application you secured and how you secured it?

6. What is your approach to mitigating the OWASP Top 10 attacks?

7. Which AWS security tools would you use to mitigate OWASP risks?

8. How do you secure AWS infrastructure?

9. How do you manage secrets in AWS?

10. How do you handle secrets in Terraform?

11. How do you secure Terraform state files in a remote team environment?

12. How do you secure CI/CD pipelines?

13. How do you integrate security scanning tools?

14. How do you manage secrets in CI/CD?

15. How can you check a Docker image for vulnerabilities and how would you fix them?

16. What are Docker image building best practices?

17. How do you secure SSH access?

18. What is SELinux and how do you troubleshoot it?

19. How do you configure auditd?
