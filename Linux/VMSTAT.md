**VMSTAT**

vmstat (Virtual Memory Statistics) is a Linux performance monitoring tool that provides a real-time view of system performance trends over time.

It shows consolidated metrics for:

CPU usage
Memory usage
Swap activity
I/O activity
Process scheduling

Why vmstat is important?

Unlike:

free → snapshot of memory
top → current process view

👉 vmstat shows behavior over time (trend analysis)
