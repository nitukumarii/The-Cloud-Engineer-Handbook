# Class Attribute

A **class attribute** is a variable that belongs to the **class itself**. It is **shared by all objects** of that class.

### Rule to Remember

- **Same for all objects → Class Attribute**
- **Different for each object → Instance Attribute (`self`)**




# Instance Attribute

An **instance attribute** is a variable that belongs to a **specific object (instance)** of a class. Every object stores its own value for that attribute using **self**.

### Rule to Remember
- **Different for each object → Instance Attribute (`self`)**
- **Same for all objects → Class Attribute**
