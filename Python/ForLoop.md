# Python For Loop Syntax

The basic syntax of a `for` loop in Python is:

```python
for variable in collection:
    # code block
```

- The `variable` can be any valid name.
- The `collection` can be a string, list, tuple, etc.

---

# Example: Iterating Over a String

```python
s = "cat"

for letter in s:
    print(letter)
```

---

# Output

```
c
a
t
```

---

# Key Idea

- A string is treated as a sequence of characters.
- The `for` loop goes through each character one by one.
