# Frequently Asked Python Screening Questions (SRE / DevOps / Cloud)

These are the **most commonly asked screening questions** in early interview rounds.

---

# 1. Python Basics (Very Common)

- What is Python?
- Difference between list, tuple, set, and dictionary?
- What are mutable and immutable types?
- What is indentation in Python?
- Difference between == and is?
- What are Python data types?
- What is type casting?

---

# 2. Strings (Very Frequently Asked)

- Reverse a string
- Check palindrome
- Count occurrences of a character
- Count vowels in a string
- Check if two strings are anagrams
- Find first non-repeating character

---

# 3. Lists / Arrays (Very Important)

- Find maximum and minimum element
- Find second largest element
- Remove duplicates from list
- Merge two lists
- Find missing number
- Rotate array
- Count occurrences of element

---

# 4. Dictionaries (Core Interview Area)

- Count frequency of elements
- Word frequency in a sentence
- Character frequency
- Find duplicate elements using dictionary

---

# 5. Loops & Logic (Always Asked)

- Difference between for loop and while loop
- Print numbers from 1 to N
- Sum of digits of a number
- Reverse a number
- FizzBuzz problem

---

# 6. Functions (Important for Scripting Roles)

- What is a function?
- Difference between function and method
- What are *args and **kwargs?
- Write a retry function
- Write a health check function

---

# 7. File Handling (VERY IMPORTANT for SRE)

- Read a file line by line
- Extract ERROR logs from file
- Count ERROR occurrences
- Parse a CSV file
- Parse a JSON file

---

# 8. Exception Handling

- What is try-except block?
- Handle division by zero error
- Why is exception handling important in production?
- What is finally block?

---

# 9. SRE / DevOps Concept Questions

- What is a health check?
- What is a retry mechanism?
- What is logging?
- Difference between monitoring and alerting?
- What is high availability?
- What is a production incident?

---

# 10. Coding Pattern Questions (Very Frequent)

- Loop through string and count characters
- Loop through list and find max/min
- Dictionary-based counting problems
- Simple log file parsing
- Basic automation scripts

---

# Most Important Takeaway

If you prepare only this list well, you will cover **80–90% of screening round Python questions** for:

- Oracle
- Amazon
- Microsoft
- Google (SRE/Infra)
- Any DevOps/SRE role

Focus more on:
- Strings
- Lists
- Dictionaries
- File handling
- Basic scripting logic
